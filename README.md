# どこ食べルーレット

現在地周辺のお店をランダムで選んでくれる、優柔不断な人のためのWebアプリケーションです。

## 概要

「今日のランチどこにしよう」と悩むあなたのために、Google Places APIを使用して現在地周辺のレストランをランダムに選んでくれるアプリです。ルーレット風の演出で、楽しくお店を決められます。

## 機能

- **ランダム選択**: 現在地周辺のレストランをランダムに選択
- **ルーレット演出**: スロットマシン風のアニメーションでお店を表示
- **Google Maps連携**: 選ばれたお店の地図とルート案内
- **シェア機能**: 決まったお店をX（Twitter）でシェア可能
- **履歴機能**: 過去に選ばれたお店を最大10件保存
- **カスタマイズ可能**: 検索範囲、ジャンル、営業中のみなどのフィルター設定

## 技術スタック

- **フレームワーク**: Next.js 14 (App Router)
- **言語**: JavaScript
- **スタイリング**: CSS Modules
- **API**: Google Maps JavaScript API / Google Places API
- **デプロイ**: Vercel（推奨）

## 使い方

1. ページを開くと位置情報の許可を求められます
2. 必要に応じて検索範囲やジャンルを設定
3. 「お店を決める！」ボタンをクリック
4. ルーレット演出の後、ランダムにお店が選ばれます
5. 「Google Mapsで見る」でルート案内、「Xでシェア」で共有可能
6. 気に入らない場合は「別のお店」で再選択
