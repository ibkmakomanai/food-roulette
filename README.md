# どこ食べルーレット

現在地周辺のお店をランダムで選んでくれるWebアプリです。

## 概要

「今日のランチどこにしよう」と悩むあなたのために、Google Places APIを使用して現在地周辺のレストランをランダムに選んでくれるアプリです。アニメーション付きの演出で、楽しくお店を決められます。

## 機能

- **ランダム選択**: 現在地周辺のレストランをランダムに選択
- **Google Maps連携**: 選ばれたお店の地図を確認
- **シェア機能**: 決まったお店をX（Twitter）でシェア可能
- **履歴機能**: 過去に選ばれたお店を保存
- **カスタマイズ可能**: 検索範囲、ジャンル、営業中のみなどのフィルター設定

## 使い方

1. 必要に応じて検索範囲やジャンルを設定
2. 「お店を決める！」ボタンをクリック （位置情報の許可を求められます）
3. ルーレット演出の後、ランダムにお店が選ばれます
4. 「Google Mapsで見る」で場所の確認、「Xでシェア」で共有可能
